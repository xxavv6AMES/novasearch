Privacy Policy

Information We Collect

When you use Nova Search, we collect:

  • Basic search queries and results interaction
  • Optional AI feature usage when enabled
  • Device and browser information
  • Approximate location (with consent, for local results)
  • Customization preferences
  • Service performance metrics

Search Data Usage

Your search data helps us:

  • Deliver relevant search results
  • Improve result rankings
  • Provide local and personalized results
  • Protect against automated misuse
  • Debug and optimize performance

Optional AI Features

When you use Astro AI features:

  • AI processing is done on-demand only
  • Responses are generated in real-time
  • No personal data used in AI training
  • AI interactions are separately stored
  • You can opt-out at any time

Data Protection

  • Regular security assessments
  • Compliance with privacy regulations
  • Transparent data handling practices
